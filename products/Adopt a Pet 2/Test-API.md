# Test API

**How to Test the user api.**

To test the external User API

Copy the API URL [http://sb-se-international.northeurope.cloudapp.azure.com:8020/users ](link)

Open Swagger Explore [https://explore.swaggerhub.com/ ](link) 

Paste URL into a Swagger Explore tab and click Send

Modify inputs (parameters, headers etc.) to experiment with the API
