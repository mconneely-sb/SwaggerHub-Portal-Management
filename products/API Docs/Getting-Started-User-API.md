# Getting Started with the User API Workflow

This **user** workflow provides a comprehensive process to utilize the User APIs, it starts by searching for available users based on criteria, then moves to select a user, confirm the selection, update the user's status, and finally confirm the user's status update. The workflow is designed to be both developer and user-friendly, providing clear steps and expected outcomes. 

Following this workflow will greatly reduce your meantime to integration 🚀
